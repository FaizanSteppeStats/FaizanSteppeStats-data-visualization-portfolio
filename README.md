# Faizan Ahmed Janjua — Data Visualization Portfolio

Five portfolio-ready projects for Data Visualization Intern, Junior Data Analyst, BI Intern, and Reporting Analyst applications.

## Projects
1. Sales Performance Dashboard
2. Portugal Housing & Rent Dashboard
3. Climate Trends Visualization
4. Streaming Content Analytics Dashboard
5. Business KPI Executive Dashboard

## Tools
Python, Pandas, NumPy, Matplotlib-ready data, HTML dashboards, CSV datasets, and Jupyter notebooks.

## How to run
```bash
pip install -r requirements.txt
```
Then open any project folder and run:
```bash
python scripts/analyze.py
```

All datasets are synthetic and created for portfolio demonstration.
